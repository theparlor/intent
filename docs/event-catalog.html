Event Catalog

Structured events for the Intent system

The event catalog defines the structured events that flow through the Intent system. These events are the building blocks of observability and allow different tools and processes to communicate.

Event Types

Events are categorized by the phase of the notice→spec→execute→observe loop they represent:

  • notice.* – Problem or opportunity identified
  • spec.* – Specification created or updated
  • execute.* – Work executed, results produced
  • observe.* – Outcomes measured and recorded
