Architectural Review Board

Perspectives of key technical personas during intent spec and execution.

🏗️
Architect
Cowork surface
  • Does this spec align with our signal model?
  • What new dependencies or contracts emerge?
  • Could this cause cascading changes?
  • Where are the boundary crossings?
⚙️
Builder
Claude Code surface
  • Can I execute this spec unambiguously?
  • What's the minimal test path?
  • Are dependencies resolvable locally?
  • What edge cases should I handle?
🔍
Observer
Signal capture
  • What signals does execution emit?
  • Are success/failure conditions clear?
  • How do we detect surprises or drift?
  • What should we log for future audit?
🎯
Product Owner
Intent validator
  • Does this solve the actual problem?
  • What's the user experience impact?
  • Are trade-offs acceptable?
  • How do we measure success?
