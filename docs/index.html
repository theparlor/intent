The Shift

What Changes

Sprint
→ Loop

Continuous Flow

No ceremony walls. Work moves based on dependency graphs, not two-week calendar blocks.

Ticket
→ Contract

Verifiable Units

Done when assertions pass, not when cards move. Agents execute against schemas, not prose.

Estimation
→ Observation

Evidence over Forecast

Don't guess how long. Observe what happened and let the system learn from its own traces.

Backlog
→ Signal Stream

Living Intake

Signals are captured continuously from work, research, and conversation — not groomed in quarterly rituals.

Standup
→ Dashboard

Ambient Visibility

OTel-compatible events make work observable at all times. No status meetings required.

Retro
→ Observe Cycle

Continuous Learning

An agent reads traces and writes findings daily. Learning is a system property, not a quarterly ceremony.

How It Works Under the Hood

Intent is file-native and git-tracked. Signals, intents, specs, contracts, decisions, and events live in a .intent/ directory inside every repo. State changes emit structured events that make work observable.
