Key Artifacts

Signal

Observations from work, research, or conversations that propose a change or opportunity.

Intent

A hypothesis-driven outcome with success criteria. Answers "why are we building this?"

Spec

Declarative behavior description. Agents and humans execute against it until assertions pass.

Contract

Verifiable interface agreement. Done when tests pass, not when cards move.

The Shift

From Sprint to Loop

Continuous notice → spec → execute → observe. No ceremony walls. Work flows based on dependency graphs, not calendar weeks.

From Tickets to Specs

Work units are declarative behavior descriptions, not units of human labor. Agents consume specs directly.

From Process to Ontology

A formal work hierarchy (Signal → Intent → Spec → Contract → Capability → Feature → Product) replaces ad-hoc ceremonies.
