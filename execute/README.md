# Execute

> Implementation artifacts — where built things live.

The Execute phase is where specs become running software. With a well-shaped spec, execution involves any combination of human and AI work. The spec is the source of truth.

Key principle: **the agent can make implementation decisions the architect didn't specify.** The spec draws the boundaries. Inside those boundaries, the agent has autonomy.
